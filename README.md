
#### Abstract
Sequentially ordered data naturally arise in a multitude of Data Science applications, thus undermining the need for scalable Machine Learning algorithms. This thesis aims to build a mathematically rigorous bridge between efficient kernel methods for sequentially ordered data and the signature transform, which is a non-commutative power series in the tensor algebra whose coefficients can be seen in close analogy to monomials. We begin with a self-contained review of the signature transform from a purely algebraic perspective and then connect these objects to kernel methods via the signature kernel, thereby generalizing polynomial kernels from static to sequentially ordered data. Additionally, we provide a more practically usable discretized version of the signature kernel.

To overcome the quadratic cost in the number of sequences incurred by Gram matrix computations, we analyse two prominent dimensionality-reduction techniques that yield asymptotically linear training time in the number of sequences. We first combine Random Fourier Features with the discretized signature kernel, which are based on a Monte-Carlo approximation of a spectral measure that arises due to Bochner's Theorem. We provide one theoretical contribution by focusing on tensor random projections; using the Khatri-Rao product of very-sparse random projections we prove pointwise convergence with mixed tails and show speed-up of order `1/2` compared to the iid Gaussian case presented in \cite{toth2025} for high-dimensional sequential data.
